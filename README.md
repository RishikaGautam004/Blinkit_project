# Blinkit_project
A Power BI dashboard provides a comprehensive sales performance analysis for Blinkit – India’s Last Minute App. It is designed to help stakeholders understand overall revenue, customer behavior, outlet performance, and product-level insights through interactive visualizations and KPIs.
